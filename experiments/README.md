# 실험 기록

## 디렉토리 규칙

```
experiments/
└── YYYY-MM-DD-주제/
    ├── README.md     # 설정 + 결과 + 분석
    └── outputs/      # 결과 파일 (그래프, 영상 등)
```

## README.md 템플릿

```markdown
# 실험: <주제>

**일시:** YYYY-MM-DD
**모델:** <모델명 및 버전>
**데이터:** <데이터셋 또는 영상>

## 설정
- 파라미터 1: 값
- 파라미터 2: 값

## 결과
- 지표 1: 값
- 지표 2: 값

## 분석
관찰 사항, 실패 케이스, 인사이트 등

## 결론
Go/No-Go 판단 또는 다음 단계
```
