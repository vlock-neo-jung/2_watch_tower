# Zone Tool 사용자 매뉴얼

Watch Tower Zone Tool은 건설 현장 영상 위에 감시 구역(Zone)을 설정하고, 해당 구역에서 발생하는 침입 이벤트를 어노테이션하는 웹 기반 도구이다.

---

## 1. 기본 개념

### 1.1 Zone (감시 구역)

Zone은 카메라 영상 위에 다각형(Polygon)으로 정의하는 **감시 영역**이다. AI 시스템은 이 영역 안으로 사람이나 물체가 진입했는지를 실시간으로 판단한다.

하나의 영상에 여러 개의 Zone을 설정할 수 있으며, 각 Zone에는 이름과 타입을 부여한다.

#### Zone 타입

| 타입 | 색상 | 용도 | 예시 |
|------|------|------|------|
| **danger** (위험) | 빨강 | 출입 금지 구역 | 크레인 작업 반경, 굴착기 선회 범위 |
| **warning** (경고) | 주황 | 주의가 필요한 구역 | 자재 적재 구역, 차량 통행로 |
| **entry** (출입) | 파랑 | 출입구 / 통과 지점 | 현장 정문, 비상구 |

### 1.2 어노테이션 (Annotation)

어노테이션은 영상의 **특정 구간에서 Zone에 침입이 발생한 사실을 기록**하는 작업이다. "프레임 150에서 200 사이에 zone_1에 사람이 진입했다"와 같은 정보를 남긴다.

이 데이터는 AI 모델의 학습 데이터(Ground Truth)나 감지 정확도 평가에 사용된다.

### 1.3 두 가지 모드

도구는 두 가지 모드로 동작한다.

| | Zone 편집 모드 | 어노테이션 모드 |
|---|---|---|
| **목적** | Zone 생성, 수정, 삭제, 저장 | 침입 이벤트 기록 |
| **Zone 그리기** | O | X |
| **Zone 꼭짓점 수정** | O | X |
| **이벤트 기록** | X | O |
| **GT 저장/불러오기** | X | O |

## 2. 화면 구성

```
┌──────────────────────────────────────────────────────┐
│  A. 상단 바                                          │
│  [모드 선택 ▼]  [영상 선택 ▼]  (GT 저장) (GT 불러오기) │
├───────────────────────────────────┬──────────────────┤
│                                   │                  │
│  B. 영상 영역                      │  C. Zone 패널    │
│  - 영상 재생기                     │  - 설정 선택     │
│  - Zone 오버레이                   │  - Zone 목록     │
│  - 재생 컨트롤                     │  - 추가/삭제     │
│                                   │  - 저장          │
│                                   │                  │
├───────────────────────────────────┴──────────────────┤
│  D. 하단 타임라인                                     │
│  - 이벤트 시각화 바                                   │
│  - [침입 시작] [침입 끝]                              │
│  - 이벤트 테이블                                      │
└──────────────────────────────────────────────────────┘
```

### A. 상단 바

- **모드 선택**: "Zone 편집" 또는 "어노테이션" 전환
- **영상 선택**: 작업할 영상 파일 선택
- **GT 저장 / GT 불러오기**: 어노테이션 모드에서만 표시

### B. 영상 영역

영상 재생기 위에 Zone이 반투명 다각형으로 겹쳐서 표시된다.

**재생 컨트롤:**

| 컨트롤 | 동작 |
|--------|------|
| ⏪ | 5초 뒤로 |
| ◀ | 1프레임 뒤로 (일시정지) |
| ▶ / ⏸ | 재생 / 일시정지 |
| ▶ | 1프레임 앞으로 (일시정지) |
| ⏩ | 5초 앞으로 |
| 0.5x / 1x / 2x / 4x | 재생 속도 |
| 슬라이더 | 원하는 시점으로 이동 |

**키보드:**

| 키 | 동작 |
|----|------|
| ← | 1프레임 뒤로 |
| → | 1프레임 앞으로 |

### C. Zone 패널 (우측 사이드바)

- **설정 선택 드롭다운**: 저장된 Zone 설정 파일을 불러온다
- **Zone 목록**: 현재 로드된 Zone 목록. 클릭하면 해당 Zone 선택
- **\[+ 추가\]**: 새 Zone 추가 (Zone 편집 모드에서만)
- **\[- 삭제\]**: 선택된 Zone 삭제 (Zone 편집 모드에서만)
- **파일명 입력 + \[저장\]**: Zone 설정을 YAML 파일로 저장 (Zone 편집 모드에서만)

### D. 하단 타임라인

- **시각화 바**: 영상 전체 길이를 막대로 표시. 이벤트 구간이 색상 블록으로 나타남
- **재생 헤드**: 현재 재생 위치를 흰색 선으로 표시
- **\[침입 시작\] / \[침입 끝\]**: 이벤트 기록 버튼 (어노테이션 모드에서만)
- **이벤트 테이블**: 기록된 이벤트 목록 (Zone, 시작/끝 시간, 길이, 삭제)

## 3. Zone 편집 모드

### 3.1 Zone 생성 (그리기)

**순서:**

1. 상단 바에서 **영상을 선택**한다
2. 모드가 "Zone 편집"인지 확인한다
3. 우측 패널에서 **\[+ 추가\]** 버튼을 클릭한다
4. 다이얼로그에서 **Zone 이름**과 **타입**(danger / warning / entry)을 입력하고 **\[확인\]**을 누른다
5. 영상 위에 안내 메시지가 표시된다:
   > 클릭: 꼭짓점 배치 | 우클릭: 완성 | Esc: 취소
6. 영상 위에서 **좌클릭**으로 다각형의 꼭짓점을 하나씩 배치한다
   - 각 꼭짓점은 파란색 원으로 표시되며 선으로 연결된다
   - 마우스를 움직이면 마지막 꼭짓점에서 점선 미리보기가 나타난다
7. 꼭짓점을 **3개 이상** 배치한 뒤, **우클릭**으로 다각형을 완성한다
8. 완성된 Zone이 타입에 맞는 색상의 반투명 다각형으로 표시되고, 우측 목록에 추가된다

> **Tip:** 꼭짓점이 3개 미만인 상태에서 우클릭하면 자동으로 취소된다. Esc 키를 눌러도 그리기를 취소할 수 있다.

### 3.2 Zone 수정 (꼭짓점 편집)

1. 우측 패널의 Zone 목록에서 수정할 **Zone을 클릭**하여 선택한다
2. 선택된 Zone의 각 꼭짓점에 **드래그 가능한 컨트롤 포인트**가 나타난다
3. 꼭짓점을 **마우스로 드래그**하여 원하는 위치로 이동한다
4. 드래그를 놓으면 변경된 좌표가 자동으로 반영된다

### 3.3 Zone 삭제

1. 우측 패널에서 삭제할 **Zone을 클릭**하여 선택한다
2. **\[- 삭제\]** 버튼을 클릭한다
3. 선택된 Zone이 즉시 제거된다

> Zone을 선택하지 않으면 삭제 버튼은 비활성 상태이다.

### 3.4 Zone 설정 저장

1. 우측 패널 하단의 파일명 입력 필드에 **저장할 파일명**을 입력한다 (예: `site_a.yaml`)
2. **\[저장\]** 버튼을 클릭한다
3. 현재 화면의 모든 Zone이 해당 이름의 YAML 파일로 서버에 저장된다

### 3.5 Zone 설정 불러오기

1. 우측 패널 상단의 **설정 선택 드롭다운**을 클릭한다
2. 저장된 설정 파일 목록에서 원하는 파일을 선택한다
3. 해당 파일에 저장된 Zone들이 화면에 로드된다

## 4. 어노테이션 모드

### 4.1 어노테이션 작업 흐름

어노테이션은 영상을 재생하면서 "이 구간에서 이 Zone에 침입이 있었다"는 사실을 기록하는 작업이다.

**사전 준비:**

- Zone이 하나 이상 설정되어 있어야 한다 (Zone 편집 모드에서 미리 생성하거나, 저장된 설정을 불러온다)

**순서:**

1. 상단 바에서 모드를 **"어노테이션"**으로 전환한다
2. 영상을 재생하면서 침입이 발생하는 시점을 찾는다
3. 우측 패널에서 해당 **Zone을 클릭**하여 선택한다
4. 침입이 시작되는 프레임에서 **\[침입 시작\]** 버튼을 클릭한다
   - 버튼이 빨간색 "🔴 재시작 (프레임번호)"로 바뀐다
   - 하단에 "기록 중..." 표시가 깜빡인다
5. 침입이 끝나는 프레임에서 **\[침입 끝\]** 버튼을 클릭한다
   - 이벤트가 생성되어 타임라인 바에 색상 블록으로 표시된다
   - 이벤트 테이블에 새 행이 추가된다
6. 같은 방식으로 영상 전체에 걸쳐 이벤트를 반복 기록한다

> **Tip:** 프레임 단위로 정밀하게 조절하려면 **◀ / ▶ 버튼** 또는 **← / → 키**를 사용한다.

### 4.2 이벤트 관리

**이벤트 확인:**
- 하단 타임라인의 색상 블록을 클릭하면 해당 이벤트의 시작 프레임으로 이동한다
- 이벤트 테이블의 행을 클릭해도 동일하게 이동한다

**이벤트 삭제:**
- 이벤트 테이블에서 해당 행의 **\[X\]** 버튼을 클릭한다

### 4.3 GT(Ground Truth) 저장

1. 상단 바의 **\[GT 저장\]** 버튼을 클릭한다
2. 팝업에서 **파일명**을 입력한다 (기본값: `gt_{영상명}.json`)
3. 현재 Zone 설정 + 모든 이벤트가 JSON 파일로 서버에 저장된다

저장되는 데이터:

- 영상 정보 (파일명, fps, 해상도, 총 프레임)
- Zone 정의 (모든 Zone의 이름, 타입, 좌표)
- 이벤트 목록 (각 이벤트의 Zone ID, 시작/끝 프레임)

### 4.4 GT 불러오기

1. 상단 바의 **\[GT 불러오기\]** 버튼을 클릭한다
2. 저장된 GT 파일 목록에서 원하는 파일을 선택한다
3. Zone과 이벤트가 모두 복원된다

## 5. 일반적인 작업 시나리오

### 시나리오 A: 처음부터 Zone 설정 만들기

```
1. 영상 선택
2. [Zone 편집] 모드
3. [+ 추가] → 이름: "크레인 반경", 타입: danger → 꼭짓점 그리기 → 우클릭 완성
4. [+ 추가] → 이름: "자재 적재소", 타입: warning → 꼭짓점 그리기 → 우클릭 완성
5. [+ 추가] → 이름: "정문", 타입: entry → 꼭짓점 그리기 → 우클릭 완성
6. 파일명 입력: "site_a.yaml" → [저장]
```

### 시나리오 B: 기존 Zone으로 어노테이션 작업

```
1. 영상 선택
2. [Zone 편집] 모드 → 설정 드롭다운에서 "site_a.yaml" 선택
3. [어노테이션] 모드로 전환
4. 영상 재생 → 침입 발견 시 Zone 선택 → [침입 시작] → [침입 끝]
5. 반복
6. [GT 저장] → 파일명 입력
```

### 시나리오 C: Zone 위치 미세 조정

```
1. [Zone 편집] 모드 → 설정 불러오기
2. 목록에서 수정할 Zone 클릭
3. 꼭짓점 드래그로 위치 조정
4. 설정 다시 저장
```

## 6. 참고 사항

### 좌표 시스템

Zone 좌표는 영상 크기에 대한 **비율(0.0 ~ 1.0)**로 저장된다. 예를 들어 좌표 `[0.5, 0.5]`는 영상의 정중앙을 의미한다. 이를 통해 화면 크기가 달라져도 Zone 위치가 정확하게 유지된다.

### Zone 설정 파일 형식

Zone 설정은 YAML 파일로 저장되며, 각 Zone에는 다음 정보가 포함된다:

| 필드 | 설명 | 기본값 |
|------|------|--------|
| `zone_id` | 고유 식별자 | 자동 생성 |
| `zone_name` | 사용자가 지정한 이름 | - |
| `zone_type` | danger / warning / entry | - |
| `geometry` | GeoJSON Polygon 좌표 (정규화) | - |
| `triggering_anchor` | 감지 기준점 | BOTTOM_CENTER |
| `target_classes` | 감지 대상 클래스 | [0] (사람) |
| `min_consecutive_frames` | 최소 연속 감지 프레임 | 3 |
| `cooldown_ms` | 재감지 대기 시간 (밀리초) | 30000 |

### 저장 위치

| 데이터 | 서버 경로 |
|--------|-----------|
| Zone 설정 (YAML) | `{DATA_ROOT}/configs/zones/` |
| GT 어노테이션 (JSON) | `{DATA_ROOT}/zone_gt/` |

`DATA_ROOT`는 기본값 `/home/neo/share/watch-tower_data`이며, 환경변수 `WATCH_TOWER_DATA_ROOT`로 변경 가능하다.
