# 모델별 영상 정성 평가 (육안 비교)

- 작성일: 2026-03-21
- 프로젝트: WatchTower (건설 현장 AI 안전 모니터링)
- 관련 이슈: FWY-61

## 1. 목적

SODA test set 정량 벤치마크(Recall 84.1%)와 달리, 실제 건설현장 CCTV 영상에서 모델이 어떻게 동작하는지 육안으로 확인한다.

## 2. 테스트 설정

| 항목 | 설정 |
|------|------|
| 스크립트 | `scripts/run_tracking.py` (Detection + ByteTrack) |
| conf 임계값 | 0.25 |
| 출력 | `outputs/tracking/{모델명}/` |

### 테스트 영상

| 영상 | 해상도 | 길이 | 시점 |
|------|--------|------|------|
| construction_helmets | 1280x720 | 8초 | 눈높이/중간 |
| construction_site_cctv | 586x480 | 87초 | 중간~높음 |
| construction_subway_cctv | 1280x720 | 201초 | 높은 조감 |
| construction_tower_crane_cctv | 1280x720 | 200초 | 높은 |
| construction_site_gate | 406x720 | 34초 | 중간 |

### 비교 모델

| 모델 | 설명 |
|------|------|
| yolo11m (COCO) | COCO pretrained 범용 모델 |
| yolo11m-construction-hazard | 건설현장 특화 11클래스 모델 |
| yolo11m-soda-person | SODA 파인튜닝 person 단일 클래스 |

## 3. 육안 평가 결과

| 영상 | COCO | hazard | SODA-ft | 최선 |
|------|------|--------|---------|------|
| helmets (눈높이) | 보통 | 좋음 | 나쁨 | **hazard** |
| site_cctv (중간~높음) | 나쁨 | 나쁨 | 나쁨 | - |
| subway_cctv (높은 조감) | 나쁨 | 나쁨 | 나쁨 | - |
| tower_crane (높은) | 나쁨 | 나쁨 | 나쁨 | - |
| site_gate (중간) | 보통 | 좋음 | 나쁨 | **hazard** |

**종합: hazard 모델이 육안 기준 가장 나음.**

## 4. 핵심 관찰

### SODA-ft는 건설현장에서 형편없다

SODA test set 정량 결과(Recall 84.1%)와 실제 건설현장 영상의 괴리가 극심하다.
SODA 도메인(도로/거리)에 과적합되어 COCO의 범용성을 잃었고, 건설현장에서는 COCO baseline보다도 못한 결과를 보인다.

### hazard 모델이 건설현장에서 가장 낫다

건설현장 PPE 데이터로 학습된 모델이라 건설현장 배경/복장 패턴에 익숙하다.
특히 눈높이~중간 시점에서 사람 + PPE를 함께 잡아내는 점이 강점이다.

### 높은 각도에서는 세 모델 모두 실패

subway_cctv, tower_crane_cctv에서는 어떤 모델도 사람을 제대로 찾지 못한다.
이 문제는 모델 교체로 해결되지 않으며, SAHI 조합 또는 고각 건설현장 데이터 파인튜닝이 필요하다.

## 5. SODA 파인튜닝 접근의 오류

FWY-61에서 SODA 파인튜닝을 선택한 논리:
```
건설현장 고각 CCTV에서 사람 못 찾음
  → SODA = "고각 감시카메라 데이터"
  → SODA로 파인튜닝하면 건설현장에서도 좋아질 것
```

실제로는:
- SODA는 도로/거리 보행자 데이터이지, 건설현장 데이터가 아님
- "고각"이라는 공통점만으로 도메인이 같다고 볼 수 없음
- 파인튜닝으로 COCO의 범용성을 잃어 오히려 역효과

**교훈: 파인튜닝 데이터는 타겟 도메인과 실질적으로 유사해야 한다. 키워드(고각)만 일치하는 것은 불충분하다.**

## 6. 현재 최선의 조합

건설현장 데이터 확보가 어려운 현 단계에서:

1. **hazard 모델을 기본으로 사용** — 건설현장에서 육안 기준 가장 나은 성능
2. **SAHI + COCO 조합 테스트 필요** — 아직 건설현장 영상에서 미테스트
3. **SODA-ft는 건설현장 서비스에 부적합** — SODA 도메인 한정 사용
