# Watch Tower - 아키텍처

## 시스템 개요

건설 현장 실시간 AI 안전 모니터링 시스템

## 구성 요소

### 영상 수집 레이어
- 웹캠(USB) / IP카메라(RTSP) 입력
- 멀티 카메라 동시 처리

### AI 추론 레이어
- PPE(개인보호구) 착용 감지
- 작업자 쓰러짐 감지
- 위험구역 침입 감지
- 구역 내 인원 탐지

### 이벤트 처리 레이어
- 이벤트 생성 및 저장
- 실시간 알림 (Push, SMS, 카카오 알림톡)

### 프레젠테이션 레이어
- 중앙 모니터링 대시보드 (웹 기반)
- 라이브 영상 + AI 오버레이
- 이벤트 타임라인 및 통계

## 데이터 흐름

```
카메라 → 영상 수집 → AI 추론 → 이벤트 처리 → 알림/대시보드
```
